## Create migrations:

php artisan make:model Album -m
php artisan make:model Genre -m
php artisan make:model Song -m
php artisan make:model Playlist -m
php artisan make:migration create_playlist_song_table
php artisan make:migration create_favorites_table
php artisan make:migration create_recently_played_table

## Create Fresh migration: {Before run migratoin because new project}

php artisan migrate:fresh


## To Rename the table:
php artisan make:migration rename_title_in_songs_table --table=song


## To make factories:
php artisan make:factory ArtistFactory --model=Artist
php artisan make:factory PlaylistFactory --model=Playlist

## To make diffrent seeders:
php artisan make:seeder GenreSeeder

## Run seeder:
php artisan db:seed

## Create Controllers

php artisan make:controller Api/ArtistController --api
